WEB CODING OVERVIEW

Traditional web developers’ coding languages

HTML

One of the two basic building blocks of a website; HTML is the actual content of a site, the text, images and the instructions for browsers on how to read this code.

CSS

The other crucial building block of a website; CSS is the styling code for the HTML content.

Java script

JavaScript is code to cause a function to happen in a browser. (e.g. to submit the contents of a form, or perhaps, to cause a modal-window to open and display a slideshow.)

Web developer’s app tools

Text Editor

Where code is written. I use Sublime Text but alternatives, like TextEdit on the Mac, or DreamWeaver are also good but Brad recommends ????? which is very good and free.

Command Line

Gulp, Node.js, Git and NPM use the Command Line to automate the various tasks ‘commanded’ of them.

GitHub

GitHub acts like a store, or repository where your projects (‘Repos’) can be safely stored and shared with colleagues and collaborators.

It also stores useful code that other people have written, and you can download, to make life simpler and more efficient, thereby avoiding the re-invention of the wheel time and again.

GitHub is just one of many alternative code storage sites available, but is recommended by Brad Schiff.

Web developer’s automation and organisation tools

Git

Git helps manage project files.

Git keeps a record of all changes made by you and any collaborators working on the same project and even at the same time. Those different versions can eventually be merged with each other and with the original master version once everyone is in agreement.

This project management takes place in conjunction with GitHub (or alternative) and uses Git on the Command Line to ‘push’, ‘pull’, ‘commit’ and ‘merge’ files.

Node.js

Node.js can turn your computer into a robot assistant performing tasks automatically to save time. It also uses the power of JavaScript codes to run back-end functions.

NPM

NPM (Node Package Management) is a store from which you can download all sorts of useful code (node-modules, jquery and css files) to make your coding easier and more efficient. This ‘shopping list’ of code is kept in a file called package.json which each website or project will have, as a record of that site’s code ‘ingredients’.

Gulp

Gulp is a building tool at the heart of all automation. It will be installed globally and locally on your computer. Gulp plug-ins are the building blocks. Gulpfile.js is where the code to implement the automated functions is written.

BEM

BEM (Block Element Modifier) is a philosophy of how to structure the building process which, along with PostCSS, will help create super organised HTML and css code for all browsers to read.
